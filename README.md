## 📝 Description
Aplikasi Catat Lari adalah aplikasi Android yang dirancang untuk membantu pengguna dalam mencatat dan memantau
aktivitas olahraga lari yang telah dilakukan. Pengguna dapat menyimpan riwayat lari seperti tanggal, jarak, dan 
durasi, sehingga dapat digunakan untuk melihat perkembangan aktivitas olahraga secara berkala dan meningkatkan konsistensi 
dalam berolahraga.

## Fitur
- Tambah data lari (tanggal, jarak, durasi)
- Edit data lari yang sudah tersimpan
- Hapus riwayat lari
- Menampilkan daftar riwayat lari


## ⚙️ Tech Stack
-Kotlin
-Model-View-ViewModel (MVVM)
-Room Database
